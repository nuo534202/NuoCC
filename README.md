# NuoCC

NuoCC is a small compiler for a subset of the C language. Supported syntax is listed in [Supported Features](docs/supported_features.md).

## Quick Start

This project uses CMake as the build system. Make sure you have CMake and a C++ compiler installed:

```bash
mkdir build
cd build
cmake ..
make
./nuocc input.c
```
